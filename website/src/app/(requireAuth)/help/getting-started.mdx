2. Create a new API key by clickng on `Create new secret key` in the API keys screen.
   ![Create new api key](/start/step-1.png)
3. Name the API key "Vibeflow" and click on `Create API Key`.
   ![Name api key](/start/step-2.png)
4. Copy the key and paste it in the Vibeflow [settings section](/settings).
   ![Copy key from OpenAI](/start/step-3.png)
   ![Paste key into Vibeflow settings](/start/step-4.png)

That's it! Now try creating a new project and start generating audio .
